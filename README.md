# GoGHIssue Tool

Simple CLI tool to manage issues on GitHub 

**Features**
* Create a new issue
* Read an issue
* Update an issue
* Close an issue

**Usage:**
* API Token is hardcoded. Change the value of TOKEN.
```bash
  go run GoGHissue.go -u <username> -r <repository>
```
